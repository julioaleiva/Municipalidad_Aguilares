# Demo estática para GitHub Pages

Este proyecto usa solo:

- `index.html`
- `styles.css`
- `script.js`

## Publicarla en GitHub Pages

1. Crear un repositorio en GitHub.
2. Subir estos archivos al branch `main`.
3. Ir a `Settings > Pages`.
4. En `Build and deployment`, elegir `Deploy from a branch`.
5. Seleccionar `main` y la carpeta `/ (root)`.
6. Guardar y esperar la publicación.

## Qué se puede mejorar después

- Reemplazar textos demo por contenido real.
- Agregar imágenes optimizadas de Aguilares.
- Conectar enlaces reales de trámites y expedientes.
- Separar noticias/obras en páginas internas si hace falta.
